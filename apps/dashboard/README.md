# @trailme/dashboard

The TrailMe control-room web app: a Next.js 15 App Router dashboard (React 19) where supervisors and org admins watch every guard's live position, fading per-guard trail, and the org-wide coverage heatmap over a selectable 5/10/15/30/60/120-minute window. Auth is cookie-based via `@supabase/ssr` — server components read the session, middleware refreshes it and redirects unauthenticated traffic to `/login`, and a client map shell (`react-map-gl` + `mapbox-gl`) owns the interactive Mapbox view. This is the M0 spine: auth functions end-to-end once Supabase + Mapbox keys are present, the map renders centered on a default location with an inert time-window selector, and the admin routes (`settings`, `invites`, `audit`, `roster`) exist as auth-gated placeholders that later milestones fill in. Copy `.env.local.example` to `.env.local`, supply your keys, then `pnpm dev`.
